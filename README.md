Ckoirama Pipeline
=================

> Status: In development, not released yet

Dev installation
----------------
1) Clone it
2) Install as a local pip
```
git clone https://github.com/ckoirama/ckpl.git
cd ckpl
pip install -e .

sudo apt-get install astrometry.net
sudo apt-get install astrometry-data-2mass-07

```

Usage
-----
```sh
$ ckpl
```
